Site Permission Groups

The Site Permission Groups table lists groups and the members of the groups for the current site collection. If you are a site owner, you can use this permissions list to assist you in cleaning up permission groups on your site. The content of the Site Permission Groups table is security trimmed according to the SharePoint permissions that you have on the current site collection. If you have Full Control permissions on the site, you can information in the table. Permission groups that have no members are not listed. If you want to search for a name in the table, press Ctrl + F on your keyboard.
